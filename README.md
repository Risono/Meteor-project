[ このゲームについて ]  
1970年代後半のアーケードゲームにあったようなレトロなシューティングゲームです。
宇宙船をコントロールし、隕石（Meteor）を破壊していきます。慣性をうまく考慮することが攻略法です。ステージをクリアしていくとボーナスゲームも遊べます。

[ 作者について ]  
定年退職後に勉強を始めたUnityを使って一人でゲームを開発しています。
Meteorは初めての作品です。
